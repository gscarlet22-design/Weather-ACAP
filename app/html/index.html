Alert history

    System

    Master switch. Turn off to pause the poll loop without uninstalling.

    VAPIX credentials

    Used for localhost VAPIX Digest auth (virtual ports + overlay). Required when your device is configured with authentication.

    Webhook on alerts

    POST a JSON payload to any URL when an alert activates or clears. Great for MQTT bridges, Slack webhooks, or calling an Axis Camera Station Pro event endpoint.

    MQTT publishing

    Publish JSON weather and alert payloads to an MQTT broker. Requires libcurl built with MQTT support (available on AXIS OS 12+).
    Scheme + host + port only — no trailing slash.
    —

    Email notifications

    Send an email when an alert activates (and optionally clears). Uses your SMTP relay; TLS/STARTTLS is attempted automatically.
    Use smtp:// for plain/STARTTLS or smtps:// for implicit TLS.
    Separate multiple recipients with commas.
    —

    Mock mode

    Skip real weather fetch and use a fake snapshot with a Tornado Warning active. Good for bench-testing VMS integrations.

    Export from one device, import into another to deploy identical settings across a fleet.
